はじめに
　DNML(Digtal Novel Markup Language)はHTMLライクなテキストを 書くことよって、簡単にデジタルノベルを作れるようにするツール です。
　オリジナルの作品はもちろん、特徴としてLeafやTacticsやアリスのゲームの二次創作を効果的に作成することができます。
　Susie-Pluginを使い、ゲームの画像を直接読み込み表示します。
　BGMもオリジナルの物を演奏できます。
　画面の文字もLeafフォントを使用できますので、止まっている画 像をみればオリジナルのゲームと見分けはつかないでしょう。
　このプログラムを使用するには各ゲームのインストールしてある PCとそのCDが必要です。
　対応OSは日本語、英語MS Windows 95/98/NTです。
　画面モードは65000色以上でプレイしてください。
　なお、このプログラムはフリー＆無保証です。
　このソフトを使用したことにより発生したいかなる損害においても 一切の責任は負いません。

使い方
Kanon,AIRで遊ぶにはいのうえひろしさんのホームページで、 Susie-Pluginをダウンロードしてください。
そののちに、IFPDT10.SPIをDNMLを同じフォルダにコピーして下さい。
 AIRとKanonは同じPluginを使いますので、AIRで遊ぶために新しくPluginを 手に入れる必要はありません。

1

　スタートメニューよりDNMLブラウザを選択します。
スタートメニューへ登録していない場合はDNMLをインストールしたフォルダより DNML.EXEを実行してください。

2

　DNMLブックフォルダの下にあるサンプルを見てください。右クリックのメニューより[本を開く]を選択するか、エクスプローラからドロップしてください。
これらサンプルは自分で二次創作を作る時の参考になります。

このプログラムの説明を見るには「チュートリアル.dnml」 を見てください。
DNMLで書かれた作品は P.A.W. にアーカイブしてあります。

3

質問は掲示板でお願いします。DNMLブックをよんだ感想や、DNMLの書き方が分からないときなど、気軽に 書き込んでください。

よくある質問とその答え
  • Q1.
    グラフィックが全く表示されません。

    そのゲームの画像を読み込むのに必要なSusie-Pluginを入れて下さい。 場所は、 P.A.W.のホームページに書いてあります。 Kanon用のプラグインは同梱していません。 また、ゲームのインストールされている位置が認識されていない 可能性があります。また、体験版や試用版が インストールされているとそれと間違えることがあります。 スタートメニューよりゲームの再検索を実行して、 各ゲームのインストールされている位置を 正しく設定してください。

  • Q2.
    ENTERタグのキーでALIGNでなくALINEと書いてあるブックがあります。 どちらが正しいのですか？

    旧バージョンでは綴りを間違えていたのでそうなってます(^^;;。
    どちらでも動きますが、当然ALIGNが正しいです。

  • Q3.
    痕でこのようなDNMLを書いてもちゃんと動きません
     < A HREF="#初音" ALT="初音と帰る">
    ・
    ・
    <A NAME="初音">
     痕ではBGMの9曲目の名前が"初音"になっています。ですからテンプレートファイル の中で
    <ALIAS SRC="初音" DEST="09.CDA">
    と設定してあるので"初音"は"09.CDA"に置換されます。
    ですから問いのDNMLは
    <A HREF="#初音" ALT="初音と帰る">
    ・
    ・
    <A NAME="09.CDA">
     となります。 ENTERタグやA NAMEタグの名前には曲名や背景の名前とかさならないようにしてください。

  • Q4.
    「環境設定.dnml」がありませんとエラーがでます。
    　　ゲームのインストールしてある場所はパソコンによって違うので「環境設定.dnml」は スタートメニューより「ゲームの再検索」 (もしくはdnml.exeと同じフォルダにあるseekgeme.exe)を選ぶと自動的に作られます。

ライセンス
    　
  1. DNMLはフリーソフトです。自由に使用してください。

  2. 使用者の作成したDNML用のデータに関して花梨は一切の権利を主張しません。 自作のデータは自由に再配布、商利用が可能です。

  3. DNML.EXE及びそのシステムの著作権は花梨が保持しております。

  4. 再配布条件

    　DNMLの普及のためにもコミケで自作のデータと配布することを推奨します(^_^)/。

    DNMLをオンラインで配布する場合は、それを無料でおこなう限り自由。 連絡は必要ありません。コミケなどで自作のデータと一緒に配る場合も同じです。 許可も不要で自由に配布できます。

    　ただし、お金を取って作品を配布する場合には、そのコピーを一つ花梨に渡さなければ なりません

　自作ゲームのシステムとして配布する場合はdnml.exeだけ配布しても結構です。
　DNMLの実行に必要なファイルはDNML.EXEのみです。レジストリの設定なども 行う必要はありません。
　dnml.exeをリネームすれば同じフォルダにある同じタイトルのDNMLブックを読み 込みます。
(例えば、DNML.EXEをMYGAME.EXEにリネームして起動すると、常に同じフォルダに あるMYGAME.DNMLを実行します。こうすればユーザーはDNML.EXEの存在を気にせずに すみます)
　セーブは最初に読み込んだDNMLブックと同じフォルダに記録します。よってCD-ROM から実行した場合はセーブできません。

　同人誌みたく印刷とかしなくてもフロッピー買ってくるだけで配布できるし、 見栄えもいいからノートパソコンとか持っていってデモすれば 結構売れると思うけど(^^;;
　スタートメニューに自作のデータを登録するようにインストールプログラム をカスタムして欲しい、DNMLのホームページでブースのCMをして欲しい、 フロッピーから起動できるように最小のファイルだけ配布したいなどの要望は気軽に連絡してください。
